Structural Testing¶

Structural testing is an approach to assess the quality of suite testing of program structures. There are three techniques addressed to this kind of test.

  • Statement coverage – It’s a metric used to calculate and measure the number of statements in the source code that have been covered.

  • Branch coverage – Measures whether all possible ramifications of each decision condition have been executed at least once.

  • Path coverage – Refers to measure whether all linearly independent paths in the program are executed at least once, paths are executable statements from entry to exit points.

In this notebook, you’ll see a practical example of the statement coverage. For more details on the other two techniques, check the references at the end of the page.

Create the files¶

Create a file called area.py:

%%writefile area.py

#area.py
from math import pi

def circle_area(r):
    if type(r) not in [int, float]:
        raise TypeError("The radius must be a non-negative real number.")
    if r < 0:
        raise ValueError("The radius must not be negative.")
    return pi*(r**2)

Writing area.py

Create a test file called tests1.py:

%%writefile tests.py

#tests.py
from math import pi
import unittest
from area import *

class TestCircleArea(unittest.TestCase): 
    def test_area(self):
        #Test area when radius >= 0
        self.assertAlmostEqual(circle_area(1), pi)

    def test_values(self):
        #Make sure value errors are raised when necessary
        self.assertRaises(ValueError, circle_area, -2)

Running in your computer¶

You need to install Coverage.py, the simplest way is with pip:

pip install coverage

Create a file to coverage called area.py.

Create a test file called tests.py.

Run the following command using command line:

coverage run -m unittest tests

To see the coverage, run:

coverage report -m

It is possible to create files that indicate which code parts were covered using html pages. Run the following command:

coverage html

Open the generated file on the path: /htmlcov/index.html to check the coverage report.
